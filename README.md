# Online Store Backend

Backend для интернет-магазина, разработанный на Django.  
Проект реализует серверную часть веб-приложения: работу с пользователями, товарами и заказами.

## Стек технологий
- Python
- Django
- Django REST Framework
- SQLite / MySQL
- HTML / CSS (базовый frontend для тестирования)

## Описание проекта
Проект представляет собой backend интернет-магазина с базовой бизнес-логикой.
Реализовано хранение данных, обработка пользовательских действий и административное управление контентом.

Backend отвечает за:
- управление пользователями
- хранение и обработку данных о товарах
- оформление и управление заказами
- административную часть

## Основной функционал

### Пользователи
- регистрация и авторизация пользователей
- разделение ролей (пользователь / администратор)
- управление профилем

### Товары и категории
- создание, редактирование и удаление товаров
- категории товаров
- отображение каталога

### Заказы
- создание заказа пользователем
- хранение состава заказа
- управление статусами заказов через админ-панель

### Администрирование
- кастомная админ-панель Django
- управление пользователями, товарами и заказами

## Структура проекта
- `accounts` — пользователи и аутентификация
- `shop_content` — товары, категории и логика магазина
- `common` — вспомогательные компоненты
- `mysite` — конфигурация проекта
- `media` — загружаемые файлы
