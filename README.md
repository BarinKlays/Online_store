# Online Store Backend API

Backend API для интернет-магазина, разработанный на Django и Django REST Framework.  
Проект реализует серверную часть приложения: пользователей, товары, заказы и бизнес-логику.

## Технологический стек
- Python
- Django
- Django REST Framework
- SQLite / MySQL
- Git

## Описание проекта
Проект представляет собой backend интернет-магазина с REST API.
API используется для взаимодействия с frontend или внешними клиентами.

Backend отвечает за:
- аутентификацию и авторизацию пользователей
- управление каталогом товаров
- оформление и обработку заказов
- бизнес-логику и работу с базой данных

## Основной функционал API

### Пользователи
- регистрация и авторизация
- роли: пользователь / администратор
- доступ к защищённым эндпоинтам

### Товары и категории
- получение списка товаров
- получение одного товара
- создание, обновление и удаление товаров (admin)
- категории товаров

### Заказы
- создание заказа пользователем
- получение списка заказов текущего пользователя
- управление статусами заказов (admin)

## Архитектура проекта
Проект разделён на логические приложения:
- `accounts` — пользователи и аутентификация
- `shop_content` — товары, категории и заказы
- `common` — вспомогательная логика
- `mysite` — конфигурация проекта

Используется Django ORM для работы с базой данных и REST API для клиент-серверного взаимодействия.

## Установка и запуск

1. Клонировать репозиторий:
```bash
git clone https://github.com/BarinKlays/Online_store.git
```
2. Создать и активировать виртуальное окружение:
```bash
python -m venv venv
source venv/bin/activate  # Linux / macOS
venv\Scripts\activate     # Windows
```

3. Установить зависимости:
```bash
pip install -r requirements.txt
```
4. Применить миграции:
```bash
python manage.py migrate
```

5. Создать суперпользователя:
```bash
python manage.py createsuperuser
```
6. Запустить сервер:
```basg
python manage.py runserver
```
